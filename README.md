# My-first-bioinformatics-exploration-using-BLAST

Author: Aastha Mani  
B.Tech Biotechnology, Graphic Era University

## Overview
This repository documents a small exploratory exercise conducted to understand how basic bioinformatics tools work.

The BRCA1 gene sequence from Homo sapiens was retrieved from the NCBI database and analyzed using BLAST to observe sequence similarity and alignment results.

## Workflow
1. Retrieve BRCA1 gene sequence from NCBI
2. Obtain nucleotide sequence in FASTA format
3. Submit sequence to BLAST
4. Analyze alignment results and graphical summary

## Outcome
The analysis showed 100% sequence identity with the Homo sapiens BRCA1 transcript variant.

## Tools Used
- NCBI Gene Database
- BLAST (Basic Local Alignment Search Tool)
